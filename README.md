Conky_Config_Marslo
===================

The configuration about Conky

##Usage
- Install sensors for get temperatur:
    <pre><code>$ sudo apt-get install lm-sensors
    </code></pre>
- Copy .conkyrc file and .conky folder in $HOME
    <pre><code>$ cp .conkyrc ~/
    $ cp -r .conky/ ~/
    <code><pre>
